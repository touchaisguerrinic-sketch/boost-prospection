Guide des consultants
L'IA détecte les signaux publics. Vous, vous connaissez les gens. C'est la combinaison des deux qui crée le flux d'opportunités.
👍
Mettez un pouce sur un prospect
Comment :
→ Allez dans Pipeline
→ Trouvez le prospect que vous connaissez
→ Cliquez 👍 et entrez votre prénom

Pourquoi :
Chaque pouce = une connexion réseau. Si vous connaissez quelqu'un, c'est vous qui faites l'intro.
📡
Signalez un cross-selling
Comment :
→ Vous êtes en mission chez un client
→ Vous détectez un besoin dans un autre périmètre
→ 1 message à Guillaume ou Clarisse

Pourquoi :
1 signal × 50 consultants = 50 leads / semaine. C'est notre avantage compétitif.
🤝
Partagez vos contacts ETI
Comment :
→ Sara & Lisa vous contactent
→ Partagez 2 contacts minimum
→ Décideurs ETI, anciens clients

Pourquoi :
50 consultants × 2 contacts = 100 leads qualifiés. Puis on scale x10.
Les règles d'or
⭐
Avant d'inviter en froid, demandez d'abord : "Qui connaît cette personne ?"
✅
Invitations LinkedIn : toujours sans note — message personnalisé APRÈS acceptation.
🎯
Un signal vérifié vaut 10 fois mieux qu'une invitation aléatoire.
🔗
Le réseau BOOST = notre avantage compétitif — chaque pouce compte.
⏰
Rituel hebdo : 15 min en plénière — qui a détecté quoi ? qui connaît qui ?
